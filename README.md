# assembly-projects

Collection of small **x86 assembly** programs built to learn low-level programming and computer architecture concepts.

## Projects

- hash-function – Custom string hash function in x86 assembly.
- diff – Mini diff tool that compares two text files.
- printf-mini – Simplified printf implementation with basic format specifiers.
- branch-predictor – Branch predictor simulator with simple prediction strategies.

Each folder contains its own README with build and usage instructions.
